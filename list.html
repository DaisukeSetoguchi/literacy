3歳児でもわかるカップやきそばの作り方

概要

1mmも料理したことがない私が、今回はカップ焼きそばを作ってみようと思います！

用意するもの

  • 東洋水産　でっかいやきそば弁当
  • お湯　870ml
  • 耐熱のコップ　200ml程度入るもの一つ
  • 青のり・紅ショウガなど　お好み

作り方

  1. フタ　を　あけ、3しゅるい　の　ふくろ　を　とりだす。
  2. ちゅうか　スープ　の　こな　は　コップ　に　いれて　おく。
  3. かやく　を　めん　の　うえ　に　あける。
  4. お湯　を　ようき　の　うちがわ　の　せん　まで　そそぐ。
  5. フタ　は　しめる。
  6. ３ふん　たったら　湯切り口　を　はがして、　お湯　を　すてる。このときに、お湯　の　いちぶ　を　ちゅうかスープ　の　こな　が　はいった　コップ　に　150ml　くらい　お湯　を　いれる。のこり　は　すてる。
  7. フタ　を　ぜんぶ　はがす。
  8. えきたい　ソース　を　かける。
  9. よく　まぜる。
  10. ふりかけ　を　かける。
  11. かんせい。

やきそば弁当について

【販売箇所】
主に北海道です。北海道に旅行に行けばほぼ確実に売ってます。コンビニでも売ってますし、スーパーでも売ってます。まれに東京など北海道以外のエリアでも買えます。北海道フェアをやっている場所や、アンテナショップ等では売っていることが多いです。
【いくらくらい？】
ペヤングと同じくらいの価格です。ペヤングと似ていて、大森やメガ盛りたるものもあります。
【他の味はある？】
あります。
  • 太麺コク甘ソース
  • 小樽あんかけ風
  • 旨塩
  • たらこ味バター風味
  • ちょい辛
  • ふつうの味大判
  • ふつうの味でっかい
どれもおいしいですが、特に旨塩はおすすめです。
